## Chome Devtools

    cmd + option + i (macos)打开控制台网络面板

[官方文档 Inspect network activity](https://developer.chrome.com/docs/devtools/network/#search)

### Header Options

在网络面板中,右键点击`Header Options`,可在网络面板中显示更多选项列。
建议添加以下选项

- Protocol：HTTP 协议版本号
- Content-Length：响应体大小
- Content-Encoding：响应体编码
